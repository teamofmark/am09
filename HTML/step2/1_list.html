ul은 unordered List의 약자.
사전적의미는 중요하지 않다.
속성값이 3개, circle, diosc, square 가 있으나, 이 역시 중요하지 않다.(css 인계)
중요한 것은 layout을 구성 할 때, navigation, board, sitemap등의 목록 형태 contents 를 표현하는 layout block 이라는 점이다.

ol ordered List의 약자.
사전적의미는 중요하지 않다.
속성값은 4개, "A, a, I, i"가 있으나, 이 역시 중요하지 않다.(css 인계)
중요한 것은 layout을 구성 할 때, 번호로 (순차적)정렬 되어야 하는 목록 형태 contents 를 표현하는 layout block 이라는 점이다.

dl

dl은 definition List의 약자.
정의형 목록인데, 사전적 의미는 중요하지 않다.
특별한 속성값은 없다.
중요한 것은 layout을 구성 할 때, 3단 형태(img,title,description) box의 list를 만들어내기 유용하다.

ul은 unordered list의 약자.
사전적 의미는 비중요.
속성값은 3개(circle, disc, square)가 있으나 외울 필요 없다.(css사용권장)
중요한 것은 layout 구성시 비순차목록형태 content를 표현할 때 사용하는 점.

ol은 ordered list의 약자.
사전적 의미는 비중요.
속성값은 4개+@(1,A,a,I,i...)등이 있으나 비중요. (css사용권장)
중요한 것은 layout 구성시 순서가 중요한 혹은 번호로 표시되야 하는 목록 형태 content에 사용.

dl은 definition list의 약자.
사전적 의미는 비중요.
특별한 속성값은 없다.
중요한 점은 한개의 title(dt)을 기준으로 보완하는 내용의 data(dd)를 추가 삽입할 때 유리하다.
dt와 dd의 사용 개수가 정해져 있지는 않으나 dt x 1 / dd x N 사용을 추천한다.

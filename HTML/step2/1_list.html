ul은 unordered list의 약자.
사전적 의미는 중요하지 않다.
중요한 것은 layout 구성시 navigation, board, sitemap등 같은형태의 반복 구조에 사용이 적합하다.

ol은 ordered list의 약자.
사전적 의미는 중요하지 않다.
중요한 것은 layout 구성시 번호로 정렬 되어야 하는 목록 형태의 content에 사용이 적합하다.

dl
definition list

dl 은 definition list의 약자.
정의형 목록인데, 사전적 의미는 중요하지 않다.
한개의 title(dt)을 기준으로 보완하는 내용의 data(dd)를 추가삽입 할 때 유리하다.
dt와 dd의 갯수는 딱히 정해지지 않았지만 dt x 1 / dd x 여러개를 추천한다.

ul은 unOrdered list의 약자.

사전적 의미는 비중요.
attr 은 3개 (circle,disc,square)가 있으나, html5 비지원.
중요한 것은 layout구성 시 순서가 바뀌어도 상관이 없는 content표현에 중요한 요소라는 점.

ol은 ordered list의 약자.

사전적 의미는 비중요.
속성값은 4개(+@) "A,a,I,i"가 있다.
중요한 것은 layout구성 시 순차 정렬되어야 하는(순서가 바뀌면 안됨) content 목록 표현 요소.

HTML
Hyper Text Markup Language
XML
Extensible Markup Language

dl은 definition list의 약자

정의형 목록인데, 마찬가지로 사전적의미는 중요하지 않다.
특별한 속성값은 없다.
한개의 title(dt)을 기준으로 보완하는 내용의 data(dd)를 추가삽입할 때 유리하다.
dt와 dd의 사용 횟수 제한이 없기 때문에 무한 사용 가능하나 dt x 1 dd x N 방식을 추천한다.

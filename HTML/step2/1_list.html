ul은 unordered list의 약자.
사전적 의미는 비중요.
속성값(attr)은 circle, disc, square가 있으나 / css 사용이 권장 된다.
중요한 것은 layout 구성시 navigation, board, sitemap등 목록 형태의 content 구성에 필요하다.

ol은 ordered list의 약자.
사전적 의미 비중요.
속성값은 4개(+@) "A, a, I, i"가 있는데 이 역시 css사용 권장. 중요한 것은 layout 구성시 번호로 정렬 되어야 하거나 순차적 나열이 필요한 content에 사용.

dl
definition list

dl 은 definition list 의 약자이다.
정의형 목록인데, 마찬가지로 사전적 의미는 중요하지 않다.
특별한 속성값은 없다.
한개의 title(dt)을 기준으로 보완하는 내용의 data(dd)를 추가삽입할 때 유리하다.
dt와 dd의 갯수가 딱히 정해지지는 않았으나 dt x 1, dd x 여러개를 추천한다.
